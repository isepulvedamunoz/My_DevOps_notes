File system structure ans its Description


1.  /boot   Contains file that is used by the boot loader
2.  /root   root user home directory. Its is not same as /
3.  /dev    System devices
4.  /etc    Configuration files
5.  /bin  ->    /usr/bin    Everyday user commands
6.  /sbin ->    /usr/sbin   System /filesystem commands
7.  /opt    Optional add-on applications (not part of OS apps)
8.  /proc   Running processes (Only existing in memory)
9.  /lib  ->    /usr/lib    C programming library files needed by commands ans apps
10. /tmp    Directory for temporary files
11. /home   Directory for user
12. /var    System logs
13. /run    System daemons that start very early (e.g system and udev) to store
14. /mnt    To mount external filesystem
15. /media  For cdrom mounts
